# Assets Directory

This directory contains all static assets for the RideCareShare website.

## Structure

```
assets/
├── images/           # Images, logos, icons
│   ├── logo.svg     # Main logo (SVG preferred for scalability)
│   ├── logo.png     # PNG fallback (if needed)
│   ├── logo-dark.svg # Dark theme logo variant (if different)
│   ├── favicon.ico   # Browser favicon
│   └── og-image.png  # Open Graph image for social media (1200x630px recommended)
└── README.md        # This file
```

## Logo Files

Place your RideCareShare logo files here:
- **SVG format** - Preferred (scalable, small file size)
- **PNG format** - Fallback (use for social media previews)
- **Multiple variants** - Light/dark theme versions if needed

## Image Optimization Guidelines

- **Logos**: SVG when possible, or PNG with transparent background
- **Photos**: WebP format preferred, JPEG fallback
- **Icons**: SVG for scalability
- **Social Media**: PNG at 1200x630px for Open Graph images

## File Naming Convention

- Use lowercase
- Use hyphens for spaces
- Be descriptive: `logo-light.svg`, `hero-background.jpg`, `feature-icon-1.svg`
